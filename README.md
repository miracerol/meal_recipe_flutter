# Meal Recipe Flutter

A sample recipe application developed with Flutter.

![Alt Text](https://i.hizliresim.com/c3hvogw.gif)

## Features
Theme Switcher - Light/Dark Mode

Search

Filter by category, area and ingredient

Share the recipe

Favorite the recipe
